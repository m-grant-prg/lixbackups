NAME
chkbckshare - check if network share is mounted for backups
SYNOPSIS
chkbckshare [OPTION]
DESCRIPTION
Checks to see if a pre-determined network samba share is mounted as a target
location for backup files.
OPTIONS
-b	(Boolean) Changes output from mount information to TRUE or FALSE

-h	prints usage information

-t	(Terse) Removes prepending scriptname from output

-v	prints version information
EXIT STATUS
0
if OK,

1
if minor problems,

64
error in command line arguments.
