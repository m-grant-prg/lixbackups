NAME
backups.conf - Configuration file for the backups suite.
DESCRIPTION
backups.conf contains one line for each parameter (Key / Value pair).
FILES
backups.conf	The parameters are as follows:-
.br
server=ServerName
.br
dir=NASBackupDirectoryAndMountDirectory
.br
user=NASUsrProfile
.br
password=NASPassword
.br
notifyuser=UserToBeNotifiedByMailOnSystemBeingBackedUp

The file is stored in an etc directory. The exact location depends on the method of installation but for Autotools installations on Linux this is likely to be /usr/local/etc/backups/backups.conf.
